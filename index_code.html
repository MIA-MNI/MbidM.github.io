MPC

MPC: Microstructural profile covariance

The microstructural profile covariance (MPC) framework models human brain networks based on similarity in microstructural profiles sampled in the direction of cortical columns (For details see Paquola et al. 2018 BioRxiv). It can be used on based on an ultra-high-resolution histological reconstruction of post-mortem brains (such as BigBrain) and via myelin-sensitive MRI data (such as the data provided by the HCP).

The detailed step by step procedure is provided on the github (click github symbol below).

Generate equivolumetric surfaces between the two cortical interfaces (using 01_constructSurfaces.sh).

These surfaces are then used for systematic intracortical intensity sampling (using 02_myelinMaptoSurf.sh).

Finally, surfToMPC.m generates MPC martrices that can be analyzed appropriately.
